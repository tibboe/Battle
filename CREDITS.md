## Recommended free sources (sourced locally — see notes)
The container that builds this repo can only reach **GitHub**; the sites below are
blocked here, so download them on your own machine and commit the files.

- **Tiny Swords** — Pixel Frog (itch.io). Free, commercial OK, credit appreciated.
  Units + terrain in one style.
- **Kenney** — kenney.nl. **CC0** (no attribution). Great terrain/tiles/backgrounds.
- **0x72 Dungeon Tileset II** — itch.io. **CC0**. Tiny animated characters + tiles.
- **Universal LPC Spritesheet Generator** — OpenGameArt / GitHub. **CC-BY-SA / GPL**
  (attribution + share-alike). Fastest way to get idle/walk/attack/death in one sheet.
- **OpenGameArt.org** — filter by **CC0 / CC-BY**.

### Licence quick-guide
- **CC0** — public domain; do anything, no credit. Safest.
- **CC-BY** — free, just credit the author.
- **CC-BY-SA / GPL** — credit *and* share-alike (may bind your derived art's licence).
- Always read each asset's own licence; "free" packs sometimes forbid redistribution.

### The tint caveat
The engine makes both armies from **one art set + azure/crimson tint**, which only
works on **light/neutral/greyscale** source art. If your chosen pack ships already
coloured units, either pick neutral art or use **two different sprites** per side and
drop the tinting (a small code change — ask and I'll wire it).
